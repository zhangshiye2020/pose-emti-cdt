# pose-emti-cdt
CLion到期，不是自己电脑不想续，改用eclipse cdt

## Ubuntu

需要ubuntu安装opencv4、pkg-config

1. `pkg-config --cflags opencv4` 找到头文件，配置eclipse的include
2. `pkg-config --libs opencv4` 找到库名称，配置eclipse的libs，关键opencv_core,opencv_imgproc,opencv_highgui,opencv_codecs
